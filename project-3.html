OpenGL Solar System

A showcase of my ongoing OpenGL programming project: Rendering a solar system.

Project Overview

My ongoing personal project is an effort to reacquaint myself with graphics programming. I first learned OpenGL programming by taking CS 480- Computer Graphics in the Fall 2021 semester at UNR. This project is an enhanced recreation of the first project from that class.

This project is currently in an alpha state with the source code publicly available in a GitHub repository linked below. There are instructions to build and run this program in a Linux environment (including dependencies) in the Readme file on that page.

Tools Used

OpenGL
GLFW
Assimp
Magick++
TinyXML2
Beta progress: texture mapping, blinn-phong lighting
Alpha progress: matrix stack, no shading
